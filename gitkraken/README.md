# gitkraken

## Setup

* Sign in with GitHub
* Open the `~/.dotfiles` repo, or clone a repo to `~/projects`
* Set `Pull (rebase)` as default Pull option (from Pull dropdown)
* Settings (leave as default if not mentioned)
  * Integrations
    * Add GitHub integration
      * On linux, gitkraken should open a your default Windows browser to create the token - copy and paste the generated token into gitkraken back in wsl
  * External Tools
    * External editor: VS Code
    * External terminal:
      * PowerShell (Windows)
  * UI Customization
    * Theme: GitKraken Dark
    * Notification location: Bottom left
  * Editor
    * Font: JetBrains Mono
    * Font size: 14
    * Word wrap: Check
  * In-App Terminal
    * Font: JetBrains Mono
    * Font size: 14
    * Default In-App terminal:
      * PowerShell (Windows)
  * Experimental
    * Git executable
      * Select installed (Windows)
